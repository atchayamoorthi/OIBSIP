# Task 4 — Research Report: Common Network Security Threats

## Objective
Write a comprehensive, well-structured research report on common network security
threats — covering how each attack works, its real-world impact, and specific
mitigation strategies.

## What This Task Covers
This task is research and documentation, not hands-on exploitation. The deliverable is
a single Markdown report, `network_security_threats_report.md`, analysing four attack
categories:
- Denial-of-Service (DoS) / Distributed Denial-of-Service (DDoS)
- Man-in-the-Middle (MITM)
- IP Spoofing
- DNS Poisoning / DNS Spoofing (bonus threat)
